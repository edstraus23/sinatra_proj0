Introduction

The Sintara web-based content management system (SWCMS) was created to make list generation and content management easy. This user guide describes how to install and use the SWCMS. It also describes the required software and documents how to run DITA Open Tool Kit.

Note: This PDF file was created using the SWCMS application.
