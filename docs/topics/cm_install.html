Installation

To install the SWCMS, use a program, like Winzip or 7-zip, to unzip the following file:

sinatra_md.zip

Unzipping the file will create a sinatra_md directory. The sinatra_md directory contains the following files at the top-level:

  • cm_app.rb - The main sintara ruby application. Starts up the sintara web application and access the datamapper database application.

  • recall.db - Pre-populated database used with the datamapper application.

There are also three directories at the top-level:

  • h2d - This directory contains file for an XSL transform that converts html files into dita.

  • public - This directory allows you to post HTML files on the Sinatra web application.

  • views - This folder contains the ERB views. See http://sinatrarb.com/intro.html for more details.

After creating the Sinatra_md directory, go to the Setup chapter to configure the SWCMS.
