Setting Variables

For cm_app.rb,  views/form1_out.erb, views/form2_out.erb,  and views/form3_out.erb, set the following variables to match your set up: ``` @sinatra_dir="/home/eric/sinatra_md1" @ditaot_dir="/home/eric/dita-ot-3.0.1" @java_dir="/usr/lib/jdk1.7.0_45" ``` or ``` @sinatra_dir="/home/eric/sinatra_md1" @ditaot_dir="/home/eric/dita-ot-3.0.1" @java_dir="/usr/lib/jdk1.7.0_45" ``` To set categories, add the different categories for the following variable: ``` cat_out=" " ``` etc
